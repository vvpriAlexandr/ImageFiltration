# Улучшение характеристик изображения для дальнейшего поиска объектов на нем
Программа на Python для для обработки изображений. Она позволяет загружать изображение, применять к нему методы обработки изображения, а также сохранять его.


## Возможности программы

### Загрузка и сохранение
- Загрузка изображения в форматах .jpg и .png
- Сохранение обработанного изображения

### Обработка изображения
- Бинаризация изображения с помощью метода Оцу
- Выделение границ с помощью алгоритма Canny
- Выделение углов с помощью алгоритма Harris
- Морфологическое закрытие
- Изменение яркости
- Изменение контраста
- Повышение резкости
- Выделять отдельные цветовые каналы изображения
- Удалять шум с помощью медианной фильтрации и фильтрации Гаусса.
- Изменение цветового пространства: в HSV или в grayscale
- Применение HSV-маски с нижним и верхним порогом
  
## Запуск
Необходимо запустить в редакторе кода файл "Proekt.py".
## Требования
- Python 3.8 +
- Установленные библиотеки: tkinter, cv2, numpy, pillow
